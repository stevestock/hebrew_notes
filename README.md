### Dora the Explorer

| ---: | :--- | :--- |

| קדימה דורה | kadima, dora! | Come on, Dora! |
| אוקיי | okay | Alright! |
| נזוז | nazuz | Let's go! |
| !דורה, דורה, דורה, החוקרת! כן | dora dora dora hachokeret! Ken! | Dora, Dora, Dora the explorer! Dora!|
